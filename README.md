# Iris Flower Classifier
A simple Machine Learning model built in Python using Scikit-Learn to classify flower species based on physical measurements.

## How it works
This project uses the **K-Nearest Neighbors (KNN)** algorithm. It achieves ~96-100% accuracy by comparing unknown flowers to the 3 closest examples it has seen before.

## Tools Used
* Python
* Scikit-Learn (ML Library)
* Google Colab
